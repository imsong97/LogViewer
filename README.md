# 📋 LogViewer
여러개 로그를 봐야하는데 한번에 하나씩만 열리는게 불편해서 만든 KMP 기반 데스크톱 애플리케이션
- **KMP (Kotlin Multiplatform) && Compose Multiplatform**
- **Architecture**: MVVM, Data Layer

---
**현재 상태**: 회사 로컬 -> 지금 repo로 코드 이동 중, 업무 중 실제 이슈 해결 시 사용중(개발완료, 테스트 단계)

## ✨ 주요 기능

### 🔗 **로그 파일 통합 관리**
- 여러 개의 로그 파일을 동시에 열기 가능
- 전체 로그를 시간별로 자동 정렬

### 🔍 **필터 기능**
- 텍스트, 태그, 로그레벨 별 필터 가능

### 📌 **북마크 기능**
- 중요한 로그에 북마크 추가
- 북마크된 로그만 별도로 표시 가능
